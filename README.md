# Java GUI Based ATM System

A desktop-based ATM simulation built using Java Swing and File Handling.
This project simulates real banking operations with a user-friendly interface.

## Features

* Secure login authentication
* Create & delete account
* Deposit & withdraw money
* Transfer funds
* Fast cash option
* Mini statement
* Export statement
* Change PIN
* Balance check
* Auto session timeout

## Technologies Used

* Java
* Java Swing
* File Handling
* Object-Oriented Programming
* Event Handling

## How to Run

Compile:

```
javac ATMGUI.java
```

Run:

```
java ATMGUI
```
